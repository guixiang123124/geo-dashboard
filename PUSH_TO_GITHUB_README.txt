================================================================================
  如何将GEO Attribution Dashboard推送到GitHub
  How to Push GEO Attribution Dashboard to GitHub
================================================================================

中文说明：
---------
1. 在GitHub上创建新仓库：
   - 访问：https://github.com/new
   - 仓库名：geo-attribution-dashboard
   - 不要勾选任何初始化选项

2. 运行推送脚本：
   Windows: push-to-github.bat
   Mac/Linux: ./push-to-github.sh

3. 输入你的GitHub用户名和个人访问令牌（PAT）
   获取PAT：https://github.com/settings/tokens

4. 查看详细说明：
   - NEXT_STEPS_CN.md（中文）
   - GITHUB_SETUP.md（中英文）

English Instructions:
--------------------
1. Create a new repository on GitHub:
   - Visit: https://github.com/new
   - Name: geo-attribution-dashboard
   - Don't initialize with any options

2. Run the push script:
   Windows: push-to-github.bat
   Mac/Linux: ./push-to-github.sh

3. Enter your GitHub username and Personal Access Token (PAT)
   Get PAT: https://github.com/settings/tokens

4. See detailed instructions in:
   - NEXT_STEPS_CN.md (Chinese)
   - GITHUB_SETUP.md (Bilingual)
